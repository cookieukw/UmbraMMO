# Boss Spritesheet Generation Prompt for Umbra Online

Generate a spritesheet for a top-down 2D MMORPG, graphics based on the MMORPG Tibia, Ultima VII, isometric angle. BOSS monsters, large and imposing creatures. White background.

No multiple angles. No same creature twice. Each boss is a single static image (no animation frames needed - the game flips the sprite horizontally for direction).

**IMPORTANT: Bosses should be LARGER and more detailed than regular mobs. These are epic boss creatures.**

## Style Notes:
- **Art Style:** Pixel art, isometric angle, Tibia/Ultima VII inspired
- **Background:** White/transparent
- **Each sprite:** Single static image facing RIGHT (game will flip horizontally)
- **Sprite canvas:** 128x128 pixels per boss (bosses should fill most of the frame - they are LARGE and imposing)
- **Theme:** Dark fantasy MMORPG called "Umbra Online" (shadow/darkness themed world)
- **Mood:** Epic, imposing, threatening - these are BOSS monsters that players fear
- **Detail Level:** Higher detail than regular mobs - more shading, more features, more intimidating
