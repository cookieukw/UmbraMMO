# Item ID Numbering Scheme

This document describes the numeric ID ranges for item files.

## Equipment

| Range | Category | Description |
|-------|----------|-------------|
| 100-199 | Headgear | Helmets, hats, crowns |
| 200-299 | Chest | Shirts, armor, robes |
| 300-399 | Pants | Leg armor, pants |
| 400-499 | Boots | Footwear |
| 500-599 | Amulets | Necklaces, pendants |
| 600-699 | Rings | Finger accessories |
| 700-799 | Backpacks | Bags, pouches |
| 800-899 | Offhand | Shields, tomes, orbs |

## Weapons

| Range | Category | Description |
|-------|----------|-------------|
| 1000-1099 | 1H Weapons | Swords, axes, maces |
| 1100-1199 | 2H Weapons | Greatswords, staves |
| 1200-1299 | Ranged | Bows, crossbows |

## Consumables

| Range | Category | Description |
|-------|----------|-------------|
| 2000-2099 | Food | Apples, bread, meat |
| 2100-2199 | Potions | Health, mana, stamina |
| 2200-2299 | Other | Scrolls, shards |

## Materials (Mob Drops)

| Range | Category | Description |
|-------|----------|-------------|
| 3000-3009 | Rat Drops | rat_tail, rat_meat |
| 3010-3019 | Slime Drops | slime_gel |
| 3020-3029 | Goblin Drops | goblin_ear, goblin_tooth |
| 3030-3039 | Wolf Drops | wolf_pelt, wolf_fang |
| 3040-3049 | Skeleton Drops | bone_fragment, skull_shard |
| 3050-3999 | Reserved | Future mob drops |

## File Naming Convention

Files are named: `{numericId}_{item_id}.json`

Example: `1000_worn_sword.json`
