## Notes for Spritesheet Artists

1. **Paperdoll layers:** Each equipment piece should be drawn as a separate layer that overlays on the base character sprite.
2. **Rarity color glow:** Consider adding a subtle colored outline/glow matching the rarity color when equipped.
3. **Starter items** (Normal rarity) should look deliberately worn, damaged, and low-quality.
4. **Shop items** (Common) should look functional and clean but not flashy.
5. **Uncommon drops** should have subtle visual flair — slightly glowing edges, unusual materials, etc.
6. **Rare drops** should look visually impressive — unique silhouettes, particle effects, magical auras.
7. **Naming convention for sprite files:** Use the same base name as the JSON file, e.g., `100_dented_helmet.png`.
